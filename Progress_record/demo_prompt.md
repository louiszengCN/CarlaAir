
以下是为 Carla-Air 仿真框架设计的 8 个宣传视频脚本 Prompt。这些 Prompt 旨在直接交付给工程 LLM（如 Claude Code 或 Codex 5.4），以生成相应的 Python 控制代码。每个 Prompt 都结合了 Carla 和 AirSim 的核心 API，展示了空地一体仿真的独特优势。

场景一：双星闪耀 (The Dual Launch)

Prompt 交付内容：
请编写一个 Python 脚本，使用 Carla-Air 框架（兼容 Carla 和 AirSim API）实现以下场景：
1. 初始化 Carla 客户端连接到本地服务器，加载 Town04 地图，设置天气为清晨（ClearMorning）。
2. 在地图的指定出生点生成一辆跑车（例如 vehicle.tesla.model3），并开启自动驾驶模式（使用 TrafficManager）。
3. 初始化 AirSim 的 MultirotorClient，连接到同一仿真环境。
4. 在跑车上方 2 米处生成一架四旋翼无人机，并调用 takeoffAsync() 使其起飞至 10 米高度。
5. 设置一个 Spectator 相机，初始视角对准跑车和无人机，随着跑车启动，相机平滑拉远，展示空地载具同时启动的宏大场景。
6. 脚本需运行 15 秒后安全退出。

预计展示效果：
清晨的阳光洒在现代化的城市街道上。一辆流线型的特斯拉跑车静静停在路边，车顶上方悬停着一架四旋翼无人机。随着引擎的轰鸣声（视觉上的启动），跑车平稳驶出，无人机同时拉升高度。镜头从特写逐渐拉远，完美呈现了 Carla 的地面交通模拟与 AirSim 的无人机控制在同一物理世界中的无缝融合。

场景二：天眼伴飞 (The Eye in the Sky)

Prompt 交付内容：
请编写一个 Python 脚本，实现无人机对地面车辆的动态伴飞：
1. 使用 Carla API 生成一辆由 TrafficManager 控制的车辆，在城市中随机行驶。
2. 使用 AirSim API 控制一架无人机，使其始终保持在车辆正后方 5 米、正上方 8 米的位置。
3. 在一个循环中，实时获取 Carla 车辆的 Transform（位置和旋转），计算无人机的目标位置。
4. 使用 AirSim 的 moveToPositionAsync() 或 moveByVelocityAsync() 接口，平滑控制无人机跟随车辆。
5. 无人机的偏航角（Yaw）需始终对准车辆的前进方向。
6. 持续跟踪 30 秒，展示平滑的跟随效果。

预计展示效果：
镜头跟随无人机的第一人称视角（或第三人称越肩视角）。下方是一辆在复杂城市街道中穿梭的汽车。无论汽车加速、减速还是急转弯，无人机都能像“天眼”一样，保持完美的相对位置和姿态进行伴飞。这展示了框架在跨平台坐标系转换和实时高频控制方面的卓越性能。

场景三：多维感知 (Multidimensional Perception)

Prompt 交付内容：
请编写一个 Python 脚本，展示空地协同的多模态传感器数据采集：
1. 车辆在街道行驶，无人机在上方悬停观察。
2. 为 Carla 车辆配置两个传感器：前视 RGB 相机（sensor.camera.rgb）和顶部 LiDAR（sensor.lidar.ray_cast）。
3. 使用 AirSim 的 simGetImages 接口，获取无人机视角的深度图（DepthPlanar）和语义分割图（Segmentation）。
4. 使用 OpenCV 或 Matplotlib 创建一个 2x2 的实时分屏显示窗口。
5. 左上角显示车辆 RGB 图像，右上角显示车辆 LiDAR 点云（投影为 2D 图像）。
6. 左下角显示无人机深度图，右下角显示无人机语义分割图。
7. 保持数据同步刷新 20 秒。

预计展示效果：
屏幕被分为四个象限，实时展示着令人惊叹的多维感知数据。上方是地面车辆的真实视觉和精准的激光雷达点云，下方是无人机俯瞰视角的深度信息和色彩斑斓的语义分割图。这一幕直观地向研究人员展示了 Carla-Air 在自动驾驶和机器人视觉算法训练中，提供丰富、同步、多模态数据的强大能力。

场景四：超视距预警 (Beyond Visual Line of Sight)

Prompt 交付内容：
请编写一个 Python 脚本，模拟无人机超视距预警场景：
1. 在 Carla 中生成一辆主车（Ego Vehicle），沿直线道路行驶。
2. 在主车前方 100 米处的盲区（如弯道后或被大车遮挡处），生成两辆发生碰撞的静止车辆，模拟交通事故。
3. 控制 AirSim 无人机在主车前方 50 米、高度 30 米处进行前置侦察。
4. 无人机通过向下看的相机检测到前方的事故车辆（可通过坐标判断或简单的图像边界框模拟）。
5. 无人机检测到危险后，立即向主车发送“减速”信号。
6. 主车接收到信号后，调用 Carla 的车辆控制接口，执行紧急制动并亮起双闪灯，停在事故现场前 20 米处。

预计展示效果：
主车正在高速行驶，前方视线被建筑物遮挡。此时，高空的无人机视角发现了前方的连环追尾事故。画面中闪烁出红色的警告信号，数据瞬间传输至地面。主车在距离事故还有很长一段距离时，便平稳地减速并停下。这生动地诠释了“空地协同”在提升自动驾驶安全性、实现超视距感知方面的巨大潜力。

场景五：风雨同舟 (Weathering the Storm)

Prompt 交付内容：
请编写一个 Python 脚本，展示动态天气系统对空地载具的全局影响：
1. 初始化车辆和伴飞的无人机，在城市中平稳行驶。
2. 初始天气设置为晴朗（ClearNoon）。
3. 5 秒后，调用 Carla 的天气控制 API，在 10 秒内平滑过渡到暴雨天气（降雨量 100，云量 100，路面积水 100）。
4. 同时调用 AirSim 的 simSetWeatherParameter，同步开启雨效和雾效。
5. 车辆开启雨刷器（如果模型支持）和前照灯。
6. 切换无人机视角为红外模式（Infrared），展示在恶劣天气下依然能清晰识别地面发热的车辆。
7. 持续展示 15 秒。

预计展示效果：
原本阳光明媚的城市瞬间乌云密布，倾盆大雨倾泻而下，路面迅速积水并反射出霓虹灯的光芒。车辆在雨中艰难前行，车灯划破雨幕。而高空的无人机在风雨中依然稳定悬停，其视角切换为红外热成像模式，穿透浓雾，清晰地锁定了下方散发着热量的汽车。这展示了框架在极端环境模拟和多传感器融合测试中的鲁棒性。

场景六：城市穿梭 (Urban Threading)

Prompt 交付内容：
请编写一个 Python 脚本，展示高难度的城市环境导航：
1. 加载一个密集的城市地图（如 Town03 或 Town05）。
2. 使用 Carla 的 TrafficManager 生成 50 辆随机行驶的背景车辆，营造繁忙的交通流。
3. 控制一辆主车在复杂的环岛或十字路口中穿梭。
4. 控制 AirSim 无人机执行一条预设的复杂样条曲线路径（Spline Path），穿过狭窄的楼宇间隙、桥洞，最后飞跃环岛中心。
5. 使用 Spectator 相机跟随无人机，展示其惊险的飞行轨迹和下方繁忙的交通。

预计展示效果：
这是一场视觉盛宴。无人机以极高的速度在摩天大楼的玻璃幕墙间穿梭，灵巧地钻过立交桥的桥洞。镜头向下俯瞰，是 Carla 引擎渲染出的极其逼真的、川流不息的城市交通。无人机的灵动与地面交通的秩序形成鲜明对比，完美展现了框架在处理高密度场景和复杂三维空间导航时的流畅度。

场景七：集群出动 (Swarm Deployment)

Prompt 交付内容：
请编写一个 Python 脚本，展示多智能体编队控制：
1. 在开阔地带（如 Town06 的高速公路旁）生成一辆大型指挥车（如 vehicle.carlamotors.carlacola）。
2. 使用 AirSim API 初始化 3 架无人机（Drone1, Drone2, Drone3）。
3. 控制 3 架无人机从指挥车周围起飞，并在空中组成一个“V”字形编队。
4. 指挥车开始向前行驶，3 架无人机保持“V”字形编队，在指挥车前方 20 米处同步伴飞。
5. 飞行 10 秒后，编队变换为“一”字形横排展开，模拟网格化搜索。
6. 持续展示 15 秒。

预计展示效果：
一辆重型指挥车停在旷野中，三架无人机如同蜂群般同时升空，迅速在空中集结成整齐的“V”字编队。随着指挥车的推进，无人机编队如影随形。随后，它们在空中优雅地变换阵型，展开成一字长蛇阵。这展示了 Carla-Air 在多智能体（Multi-Agent）协同、集群控制和复杂编队算法验证方面的强大能力。

场景八：日落归途 (Sunset Return)

Prompt 交付内容：
请编写一个 Python 脚本，实现高精度的动态降落：
1. 设置天气为壮丽的日落（Sunset），光影柔和。
2. 车辆在一条笔直的沿海公路或长直道上以 20 km/h 的恒定速度行驶。
3. 无人机在车辆后方 30 米、高度 20 米处跟随。
4. 启动降落程序：无人机计算车辆的实时速度和位置，规划一条动态降落轨迹。
5. 无人机逐渐降低高度并加速，精准地降落在正在行驶的车辆顶部。
6. 降落完成后，调用 landAsync() 锁定状态。
7. 镜头拉远，展示载着无人机的车辆驶向夕阳。

预计展示效果：
在绝美的日落余晖中，一辆汽车在公路上平稳行驶。后方的无人机开始缓缓下降，它不断调整姿态，与移动中的汽车保持完美的速度同步。最终，无人机轻盈而精准地降落在车顶。镜头逐渐拉远，汽车载着无人机驶向金色的地平线，画面渐渐变暗。这不仅展示了极其精确的相对运动学计算，也为整个宣传视频画上了一个充满科技感与诗意的完美句号。
